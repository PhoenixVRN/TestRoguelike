═══════════════════════════════════════════════════════════════
📝 ШАГ 1: СОЗДАТЬ HP БАР
═══════════════════════════════════════════════════════════════

Откройте Assets/Prefabs/Character.prefab

В Hierarchy:

Character (корневой UI объект)
├── Image (спрайт персонажа)
└── HealthBar ← СОЗДАТЬ! (ПКМ на Character → UI → Image)
    ├── Background ← СОЗДАТЬ! (ПКМ на HealthBar → UI → Image)
    └── Fill ← СОЗДАТЬ! (ПКМ on HealthBar → UI → Image)

═══════════════════════════════════════════════════════════════
📝 ШАГ 2: НАСТРОИТЬ HealthBar (родительский объект)
═══════════════════════════════════════════════════════════════

HealthBar → Inspector:

Rect Transform:
┌─────────────────────────────────────────────────────────────┐
│ Anchors: Top-Center ⭐                                      │
│ Pivot: (0.5, 0)                                             │
│                                                             │
│ Pos X: 0                                                    │
│ Pos Y: 50 (над персонажем)                                 │
│ Width: 80                                                   │
│ Height: 8                                                   │
└─────────────────────────────────────────────────────────────┘

Image:
  Color: полностью прозрачный (A=0) или удалите компонент

Add Component → Health Bar (Script)

═══════════════════════════════════════════════════════════════
📝 ШАГ 8: ПОВТОРИТЬ ДЛЯ ВРАГА
═══════════════════════════════════════════════════════────════

БЫСТРЫЙ СПОСОБ:
1. Скопируйте HealthBar из Character (Ctrl+C)
2. Откройте Enamy.prefab
3. Выберите Enamy (корневой объект)
4. Вставьте (Ctrl+V)
5. Enamy → CharacterController → Health Bar: HealthBar
6. Apply All
