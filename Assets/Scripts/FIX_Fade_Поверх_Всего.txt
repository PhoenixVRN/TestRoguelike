═══════════════════════════════════════════════════════════════
    🔧 ИСПРАВЛЕНИЕ: FADE ПАНЕЛЬ ПОВЕРХ ПЕРСОНАЖЕЙ
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА:
  Персонажи Order in Layer: 1
  Fade Panel Order in Layer: 0
  → Персонажи видны через затемнение! ❌

РЕШЕНИЕ:
  Установить у Fade Panel максимальный Order in Layer!

═══════════════════════════════════════════════════════════════
⚡ БЫСТРОЕ РЕШЕНИЕ (30 СЕКУНД)
═══════════════════════════════════════════════════════════════

1. Hierarchy → ScreenFadePanel

2. Inspector → Canvas (компонент):

   Если НЕТ Canvas:
     Add Component → Canvas
   
   Настройки:
   ┌────────────────────────────────────────┐
   │ Canvas                                 │
   │   Override Sorting: ✓  ← ВКЛЮЧИТЬ!   │
   │   Sort Order: 1000  ← МАКСИМУМ!       │
   └────────────────────────────────────────┘

3. ГОТОВО! ✅

Теперь Fade Panel рисуется ПОВЕРХ ВСЕГО!

═══════════════════════════════════════════════════════════════
🔍 АЛЬТЕРНАТИВНЫЙ СПОСОБ
═══════════════════════════════════════════════════════════════

Если у ScreenFadePanel уже есть Canvas Renderer:

Inspector → найдите параметр:
  Additional Shader Channels: ...
  Material: ...
  
Но проще использовать Canvas с Override Sorting!

═══════════════════════════════════════════════════════════════
📊 ПОРЯДОК ОТРИСОВКИ
═══════════════════════════════════════════════════════════════

Order in Layer (чем больше = рисуется позже = поверх):

  0 - Фон
  1 - Персонажи ← сейчас
  2 - UI элементы
  ...
  1000 - Fade Panel ← должен быть ТУТ!

═══════════════════════════════════════════════════════════════
🧪 ТЕСТ
═══════════════════════════════════════════════════════════════

1. ScreenFadePanel → Canvas → Override Sorting: ✓
2. Sort Order: 1000
3. Play
4. Нажмите на ScreenFadePanel → Screen Fade → ПКМ
5. Найдите тестовый метод (если добавим)
6. Или просто победите и смотрите fade!

Fade должен затемнить ВСЁ, включая персонажей! ✅

═══════════════════════════════════════════════════════════════
💡 ЕСЛИ ВСЁРАВНО НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════

Проверьте структуру:

Canvas (главный, Screen Space)
├── Game (персонажи, Order: 1)
└── ScreenFadePanel ⭐
    └── Canvas (Override Sorting: ✓, Sort Order: 1000)

ScreenFadePanel должен быть ВНУТРИ главного Canvas,
но иметь свой собственный Canvas компонент!

═══════════════════════════════════════════════════════════════
✅ ГОТОВО!
═══════════════════════════════════════════════════════════════

Теперь Fade Panel затемняет ВСЁ! 🌑
