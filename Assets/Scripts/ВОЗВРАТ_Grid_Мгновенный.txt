    ⚡ МГНОВЕННЫЙ ВОЗВРАТ GRID ПОСЛЕ ПОБЕДЫ
═══════════════════════════════════════════════════════════════

После победы Grid и UI МГНОВЕННО возвращаются на место!

═══════════════════════════════════════════════════════════════
🎬 КАК ЭТО РАБОТАЕТ
═══════════════════════════════════════════════════════════════

ШАГ 1: При старте игры
  Grid Position: X = 959 (начальная)
  → BattleManager сохраняет эту позицию 💾

ШАГ 2: Игрок расставляет героев
  Герои на позициях: (0,0), (1,1), (2,0)
  → BattleManager запомнит позиции 💾

ШАГ 3: Нажатие START
  Grid → X = 0 (двигается влево)
  ObjectToMove → X = 0 (тоже двигается)
  Враги спавнятся → Бой! ⚔️

ШАГ 4: Победа (все враги мёртвы)
  Пауза 2 секунды 🎉
  
  Grid → X = 959 МГНОВЕННО! ⚡
  ObjectToMove → начальная позиция МГНОВЕННО! ⚡
  
  Пауза 0.2 сек
  Все персонажи удаляются 🗑️
  
  Пауза 0.3 сек
  Герои респавнятся: 👤(0,0) 👤(1,1) 👤(2,0) ♻️

ШАГ 5: Готово!
  Можно начинать следующую волну! 🔄

═══════════════════════════════════════════════════════════════
📋 НАСТРОЙКИ В BATTLE MANAGER
═══════════════════════════════════════════════════════════════

BattleManager → Inspector:

Ссылки:
  Grid Manager: Grid
  Grid Transform: Grid (тот же)
  Object To Move Back: Game ⭐ (объект который в GameManager)
  Hero Prefab: Character

Настройки победы:
  Victory Delay: 2          ← Пауза после победы
  Return Speed: НЕ ИСПОЛЬЗУЕТСЯ (возврат мгновенный)
  Respawn Delay: 0.5        ← Задержка между героями

═══════════════════════════════════════════════════════════════
🔍 КАК НАЙТИ Object To Move Back
═══════════════════════════════════════════════════════════════

GameManager → Inspector → Object To Move: [Game]
                                           ^^^^
                                      ЭТО ОБЪЕКТ!

Перетащите ЭТОТ ЖЕ объект в BattleManager → Object To Move Back

Обычно это объект "Game" из Hierarchy:
  Canvas → Game ← ВОТ ОН!
