⚡ БЫСТРЫЙ СТАРТ (5 МИНУТ)
═══════════════════════════════════════════════════════════════

Если хотите БЫСТРО начать:

1. Откройте Unity и сцену Game.unity

2. Проверьте конфиги в Assets/Config/:
   
   ✓ EnamyWarrior.asset → Team: 1 ⭐
   ✓ Warrior.asset → Team: 0

3. Проверьте префабы в Assets/Prefabs/:
   
   ✓ Enamy → CharacterController → Config: EnamyWarrior
   ✓ Character → CharacterController → Config: Warrior

4. Настройте NewWave.asset:
   
   ✓ Enemies Size: 3
   ✓ Все элементы → Enemy Prefab: Enamy
   ✓ Grid Position: (7,5), (8,3), (9,7)

5. Создайте/настройте WaveSpawner в Hierarchy:
   
   ✓ Grid Manager: Grid
   ✓ Waves[0]: NewWave
   ✓ Show Debug: ✓

6. Проверьте GameManager:
   
   ✓ Wave Spawner: WaveSpawner

7. ТЕСТ:
   
   Play → Поставьте героя → WaveSpawner → ПКМ → 
   "Заспавнить текущую волну" → Враги появились? ✅

В Play режиме:

1. Hierarchy → WaveSpawner → ПКМ на компоненте:
   - "Заспавнить текущую волну" → Спавнит врагов
   - "Очистить всех" → Удаляет всех врагов

2. Console (включите Show Debug):
   - Увидите логи спавна
   - Увидите логи боя
   - Увидите ошибки если что-то не так

3. Scene View (в Play режиме):
   - Красные окружности = радиус атаки
   - Жёлтые линии = кто на кого нацелен
