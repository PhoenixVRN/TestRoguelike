ПРОБЛЕМА:
  Кнопка START появлялась сразу, потому что система считала
  ВСЕХ персонажей на поле (включая врагов с автоспавном).

РЕШЕНИЕ:
  Теперь система считает ТОЛЬКО игроков (Team 0)!

═══════════════════════════════════════════════════════════════
✅ ЧТО ИЗМЕНИЛОСЬ
═══════════════════════════════════════════════════════════════

GameManager.CheckHeroCount() теперь:

1. Проходит по всем персонажам на поле
2. Проверяет их Team
3. Считает ТОЛЬКО тех у кого Team = 0 (игроки)
4. Игнорирует врагов (Team = 1)

═══════════════════════════════════════════════════════════════
🎮 КАК ЭТО РАБОТАЕТ ТЕПЕРЬ
═══════════════════════════════════════════════════════════════

Сценарий 1: Враги заспавнились автоматически (Spawn On Start)
  1. При старте игры враги появляются
  2. heroCount = 0 (враги не считаются)
  3. Кнопка START СКРЫТА ✅
  4. Игрок расставляет своих героев
  5. heroCount > 0 (только игроки)
  6. Кнопка START ПОЯВЛЯЕТСЯ ✅

Сценарий 2: Враги спавнятся через кнопку START
  1. Игрок расставляет героев
  2. heroCount > 0
  3. Кнопка START появляется
  4. Игрок нажимает START
  5. Враги спавнятся (не влияют на heroCount)
  6. Бой начинается ✅

═══════════════════════════════════════════════════════════════
🧪 ТЕСТ
═══════════════════════════════════════════════════════════════

1. WaveSpawner → Spawn On Start: ✓ (включить)
2. Play
3. Враги заспавнятся автоматически
4. Кнопка START СКРЫТА ✅
5. Поставьте своего героя (ЛКМ на ячейку)
6. Кнопка START ПОЯВЛЯЕТСЯ ✅
7. Нажмите START → Бой! ⚔️
