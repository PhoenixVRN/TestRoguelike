═══════════════════════════════════════════════════════════════
    БЫСТРЫЙ СТАРТ - Система размещения UI на ромбовидной сетке
═══════════════════════════════════════════════════════════════

📋 ШАГ 1: НАСТРОЙКА CANVAS
────────────────────────────────────────────────────────────────
1. Hierarchy → ПКМ → UI → Canvas
2. Canvas → Canvas Scaler:
   - UI Scale Mode: Scale With Screen Size
   - Reference Resolution: 1920 x 1080

📋 ШАГ 2: СОЗДАНИЕ СЕТКИ
────────────────────────────────────────────────────────────────
1. Canvas → ПКМ → UI → Image
2. Переименуйте в "GridImage"
3. Inspector → Image:
   - Source Image: перетащите "grid" из Assets/Art
4. RectTransform:
   - Anchors: Center-Center
   - Width/Height: подберите под размер сетки
   - Position: 0, 0, 0

📋 ШАГ 3: ДОБАВЛЕНИЕ СКРИПТА
────────────────────────────────────────────────────────────────
1. Выберите GridImage
2. Add Component → Advanced Grid Placer (рекомендуется)
   ИЛИ Grid Placer (базовая версия)

📋 ШАГ 4: СОЗДАНИЕ ПРЕФАБА ДЛЯ РАЗМЕЩЕНИЯ
────────────────────────────────────────────────────────────────
1. Canvas → ПКМ → UI → Image
2. Переименуйте в "Tile"
3. Настройте внешний вид:
   - Source Image: ваша картинка
   - Width/Height: размер тайла
   - Color: цвет
4. Перетащите "Tile" из Hierarchy в Assets/Prefabs
5. Удалите "Tile" из Hierarchy

📋 ШАГ 5: НАСТРОЙКА ПАРАМЕТРОВ
────────────────────────────────────────────────────────────────
Выберите GridImage → Inspector → Advanced Grid Placer:

⚙️ НАСТРОЙКИ СЕТКИ:
   Diamond Width: 64    ← измерьте на спрайте grid.png
   Diamond Height: 32   ← обычно половина ширины

⚙️ ПРЕФАБ:
   Prefab To Place: перетащите префаб "Tile" сюда

⚙️ КОНТЕЙНЕР:
   Container: оставьте пустым (автоматически)

⚙️ НАСТРОЙКИ:
   Offset X: 0         ← подстройте если нужно
   Offset Y: 0         ← подстройте если нужно
   Check Occupancy: ✓  ← запретить дубли
   Right Click To Remove: ✓ ← ПКМ удаляет
   Show Debug: ✓       ← включите для отладки

📋 ШАГ 6: ТЕСТИРОВАНИЕ
────────────────────────────────────────────────────────────────
1. Нажмите Play ▶️
2. Кликните ЛКМ на сетку → префаб появится в центре ромба
3. Кликните ПКМ на префаб → он удалится

═══════════════════════════════════════════════════════════════

🔧 ДОПОЛНИТЕЛЬНО: ВИЗУАЛИЗАТОР СЕТКИ
────────────────────────────────────────────────────────────────
Если сложно настроить размеры:

1. GridImage → Add Component → Grid Visualizer
2. Настройте параметры:
   - Diamond Width: 64
   - Diamond Height: 32
   - Show Visualization: ✓
   - Show Coordinates: ✓
3. Нажмите Play ▶️
4. Увидите точки в центрах ромбов и координаты
5. Подстройте Width/Height и Offset пока не совпадут с grid.png
6. Выключите Show Visualization когда закончите

═══════════════════════════════════════════════════════════════

📐 КАК ИЗМЕРИТЬ РАЗМЕРЫ РОМБА
────────────────────────────────────────────────────────────────
1. Откройте grid.png в любом редакторе (Paint, Photoshop, etc)
2. Измерьте расстояние:
   
   Diamond Width = от левого угла ◄───► до правого угла
   
        ╱╲
       ╱  ╲     
      ╱    ╲    Diamond Height = от верхнего ▲
     ◄──────►                      до нижнего ▼
      ╲    ╱    
       ╲  ╱     
        ╲╱
   
3. Введите эти значения в инспекторе

Стандартные значения для изометрии:
- Width = 64, Height = 32 (соотношение 2:1)
- Width = 128, Height = 64
- Width = 100, Height = 50

═══════════════════════════════════════════════════════════════

❓ ПРОБЛЕМЫ И РЕШЕНИЯ
────────────────────────────────────────────────────────────────

❌ Клики не работают
   ✅ Проверьте что Image.Raycast Target включен
   ✅ Проверьте наличие EventSystem в сцене
   ✅ Проверьте наличие GraphicRaycaster на Canvas

❌ Префабы не в центре ромбов
   ✅ Используйте GridVisualizer для настройки
   ✅ Подберите правильные Diamond Width/Height
   ✅ Используйте Offset X/Y для подстройки

❌ Сетка смещена
   ✅ Установите Anchor и Pivot в Center (0.5, 0.5)
   ✅ Измените Offset X и Offset Y
   ✅ Проверьте Position GridImage = (0, 0, 0)

═══════════════════════════════════════════════════════════════

🎮 УПРАВЛЕНИЕ
────────────────────────────────────────────────────────────────
ЛКМ (Левая кнопка мыши) - разместить префаб
ПКМ (Правая кнопка мыши) - удалить префаб

═══════════════════════════════════════════════════════════════

💡 СОВЕТЫ
────────────────────────────────────────────────────────────────
• Включите Show Debug для логов в Console
• Используйте GridVisualizer только для настройки, потом удалите
• Можно создать несколько разных префабов и переключать их
• RectTransform префаба должен иметь Pivot в центре (0.5, 0.5)
• Для лучшей производительности отключите Raycast Target у префабов
