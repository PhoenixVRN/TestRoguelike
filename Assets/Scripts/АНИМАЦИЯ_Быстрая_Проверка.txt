Код УЖЕ вызывает анимацию! Проверьте настройки префаба:

═══════════════════════════════════════════════════════════════
✅ ЧЕКЛИСТ
═══════════════════════════════════════════════════════════════

Character.prefab:

□ Animator (компонент) есть?
  └─ Controller: Character ⭐

□ Character Animator (компонент) есть?
  └─ Show Debug: ✓

□ Character Controller (компонент):
  └─ Character Animator: назначен ⭐

═══════════════════════════════════════════════════════════════
🎬 ANIMATOR CONTROLLER (Character.controller)
═══════════════════════════════════════════════════════════════

Откройте Assets/Animation/Character.controller

Проверьте Parameters:
  □ Idle (Trigger)
  □ Move (Trigger)
  □ Attack (Trigger) ⭐
  □ Death (Trigger)

Проверьте States:
  □ Idle → Motion: IdleCharacter
  □ Move → Motion: MoveCharacter
  □ Attack → Motion: AttackCharacter ⭐
  □ Death → Motion: DeathCharacter

Проверьте Transitions:
  □ Idle → Attack (условие: Attack trigger)
  □ Attack → Idle (Has Exit Time: ✓)

═══════════════════════════════════════════════════════════════
🧪 ТЕСТ
═══════════════════════════════════════════════════════════════

1. Character → Character Animator → Show Debug: ✓
2. Play
3. Console должен показать:
   "Warrior: Attack анимация" ⭐

Если видите этот лог → код работает!
Если анимация не играет → проверьте Animator Controller
