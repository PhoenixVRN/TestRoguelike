🔍 ШАГ 1: ПРОВЕРЬТЕ CONSOLE (КОНСОЛЬ)
═══════════════════════════════════════════════════════════════

1. Внизу Unity найдите вкладку "Console"
2. Кликните на неё
3. Что вы видите?

┌─────────────────────────────────────────────────────────────┐
│ Console                                     Clear  Collapse │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ ЧТО ТАМ НАПИСАНО?                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘

ВАРИАНТ А: Видите КРАСНЫЕ сообщения (ошибки)?
   → Скопируйте текст ошибки и читайте раздел "ЧАСТЫЕ ОШИБКИ"

ВАРИАНТ Б: Видите ЖЁЛТЫЕ сообщения (предупреждения)?
   → Читайте что там написано, часто там подсказка

ВАРИАНТ В: НЕ ВИДИТЕ ВООБЩЕ НИЧЕГО?
   → Значит метод спавна не вызвался, идите к ШАГ 2

ВАРИАНТ Г: Видите логи типа "Спавн волны..."?
   → Отлично! Идите к ШАГ 3

═══════════════════════════════════════════════════════════════
🔍 ШАГ 2: ПРОВЕРЬТЕ ЧТО SPAWNER НАСТРОЕН
═══════════════════════════════════════════════════════════════

1. Hierarchy → WaveSpawner (кликнуть)
2. Inspector → Wave Spawner (Script)

ПРОВЕРЬТЕ:

□ Grid Manager: НАЗНАЧЕН? (должен быть Grid)
  ┌─────────────────────────────────────┐
  │ Grid Manager: Grid ✓                │ ← ДОЛЖНО БЫТЬ!
  └─────────────────────────────────────┘
  
  Если пусто (None):
  1. Найдите в Hierarchy объект "Grid"
  2. Перетащите его в поле Grid Manager

□ Waves: ЗАПОЛНЕН? (хотя бы 1 элемент)
  ┌─────────────────────────────────────┐
  │ Waves                               │
  │   Size: 1 ✓                         │ ← Хотя бы 1!
  │   Element 0: NewWave ✓              │ ← Конфиг назначен!
  └─────────────────────────────────────┘
  
  Если Size: 0:
  1. Измените Size на 1
  2. Перетащите NewWave.asset в Element 0

□ Show Debug: ВКЛЮЧЕН? (для диагностики)
  ┌─────────────────────────────────────┐
  │ Show Debug: ✓                       │ ← ПОСТАВЬТЕ ГАЛОЧКУ!
  └─────────────────────────────────────┘

🔍 ШАГ 4: ПРОВЕРЬТЕ ЧТО GRID СУЩЕСТВУЕТ
═══════════════════════════════════════════════════════════════

1. Hierarchy → найдите объект "Grid"
2. Разверните его (кликните ▶)
3. Внутри должны быть GridCell объекты

┌─────────────────────────────────────────────────────────────┐
│ Hierarchy                                                   │
├─────────────────────────────────────────────────────────────┤
│ ▼ Canvas                                                    │
│   ▼ Game                                                    │
│     ▼ Grid  ← ЭТОТ ОБЪЕКТ                                  │
│       • GridCell (0,0)                                      │
│       • GridCell (0,1)                                      │
│       • GridCell (1,0)                                      │
│       • ... (должно быть много!)                           │
└─────────────────────────────────────────────────────────────┘

Если Grid пуст или нет GridCell:
→ Сетка не создана! Нужно создать ячейки сначала.

═══════════════════════════════════════════════════════════════
🔍 ШАГ 5: ПРОВЕРЬТЕ КООРДИНАТЫ
═══════════════════════════════════════════════════════════════

В NewWave.asset вы указали Grid Position, например (7, 5).

Проверьте что такие ячейки СУЩЕСТВУЮТ:

1. Hierarchy → Grid → разверните
2. Найдите GridCell с координатами (7, 5)

Как найти:
- Кликните на любой GridCell
- Inspector покажет Grid Position

┌─────────────────────────────────────────────────────────────┐
│ Inspector                                                   │
│ GridCell                                                    │
│                                                             │
│ Grid Cell (Script)                                          │
│   Grid Position:                                            │
│     X: 7  Y: 5  ← ЭТИ КООРДИНАТЫ                          │
└─────────────────────────────────────────────────────────────┘

Проверьте что существуют ячейки:
□ (7, 5) - есть?
□ (8, 3) - есть?
□ (9, 7) - есть?

Если НЕТ таких координат:
→ Измените Grid Position в NewWave на существующие!
→ Например (0,0), (1,1), (2,2)

🔍 ШАГ 6: ТЕСТ С SHOW DEBUG
═══════════════════════════════════════════════════════════════

Включите отладку и попробуйте снова:

1. WaveSpawner → Show Debug: ✓
2. GameManager → Show Debug: ✓ (если есть)
3. Нажмите ▶ PLAY

5. Смотрите Console

Должны появиться логи:
┌─────────────────────────────────────────────────────────────┐
│ Console                                                     │
├─────────────────────────────────────────────────────────────┤
│ ╔═══════════════════════════════════════╗                  │

│ ║ Врагов: 3                             ║                  │
│ ╚═══════════════════════════════════════╝                  │
│ ✅ Враг заспавнен в (7, 5)                                 │
└─────────────────────────────────────────────────────────────┘

Если видите "⚠️ Не удалось заспавнить":
→ Читайте причину в логе

═══════════════════════════════════════════════════════════════
❌ ЧАСТЫЕ ОШИБКИ И РЕШЕНИЯ
═══════════════════════════════════════════════════════════════

ОШИБКА: "GridManager не найден!"
РЕШЕНИЕ:
  1. WaveSpawner → Grid Manager: назначьте Grid
  2. Или создайте Grid если его нет

ОШИБКА: "Волна 0 не существует!"
РЕШЕНИЕ:
  1. WaveSpawner → Waves → Size: должен быть хотя бы 1
  2. Element 0: назначьте NewWave.asset

ОШИБКА: "Префаб врага не назначен! Пропускаем..."
РЕШЕНИЕ:
  1. NewWave.asset → Enemies → Element 0
  2. Enemy Prefab: перетащите Enamy

ОШИБКА: "Ячейка (7, 5) не найдена!"
РЕШЕНИЕ:
  1. Такой ячейки не существует в Grid
  2. Измените Grid Position на существующие координаты
  3. Или создайте больше ячеек

ОШИБКА: "Ячейка (7, 5) уже занята!"
РЕШЕНИЕ:
  1. В этой ячейке уже кто-то стоит
  2. Выберите другие координаты
  3. Или очистите: WaveSpawner → ПКМ → "Очистить всех"

ОШИБКА: NullReferenceException: Object reference not set...
РЕШЕНИЕ:
  1. Что-то не назначено (Grid Manager, Waves, или Prefab)
  2. Проверьте все поля в Inspector
  3. Смотрите в какой строке ошибка (кликните на ошибку)
