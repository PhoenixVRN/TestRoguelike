🔍 ПРОВЕРКА #1: EventSystem
────────────────────────────────────────────────────────────────
❓ Есть ли EventSystem в сцене?

✅ ПРОВЕРИТЬ:
   Hierarchy → найдите "EventSystem"
   Если нет → Hierarchy → ПКМ → UI → Event System

❌ БЕЗ НЕГО КЛИКИ НЕ РАБОТАЮТ!

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #2: Canvas настроен?
────────────────────────────────────────────────────────────────
❓ Есть ли GraphicRaycaster на Canvas?

✅ ПРОВЕРИТЬ:
   1. Выберите Canvas в Hierarchy
   2. Inspector → должен быть компонент "Graphic Raycaster"
   3. Если нет → Add Component → Graphic Raycaster

❌ БЕЗ НЕГО КЛИКИ ПО UI НЕ РАБОТАЮТ!

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #3: GridManager настроен?
────────────────────────────────────────────────────────────────
❓ Назначен ли префаб?

✅ ПРОВЕРИТЬ:
   1. Выберите объект Grid (с компонентом GridManager)
   2. Inspector → Grid Manager
   3. Prefab To Place: должен быть назначен префаб! ⭐⭐⭐
   
❌ ЕСЛИ ПУСТО → НИЧЕГО НЕ ЗАСПАВНИТСЯ!

🔧 КАК НАЗНАЧИТЬ:
   Перетащите префаб из Project в поле "Prefab To Place"

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #4: GridCell правильно настроен?
────────────────────────────────────────────────────────────────
❓ У GridCell есть Image и он может получать клики?

✅ ПРОВЕРИТЬ:
   1. Выберите любой GridCell в Hierarchy
   2. Inspector → должны быть компоненты:
      ✓ Rect Transform
      ✓ Canvas Renderer
      ✓ Image ⭐
      ✓ Grid Cell (Script)
   
   3. Image компонент:
      ✓ Color: любой (например зеленый)
      ✓ Raycast Target: ВКЛЮЧЕН ⭐⭐⭐

❌ ЕСЛИ Raycast Target ВЫКЛЮЧЕН → КЛИКИ НЕ РАБОТАЮТ!

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #5: GridCell внутри Grid?
────────────────────────────────────────────────────────────────
❓ GridCell является дочерним объектом Grid?

✅ ПРОВЕРИТЬ:
   Hierarchy должна выглядеть так:
   
   Canvas
   └── Grid (GridManager тут!)
       ├── GridCell
       ├── GridCell
       └── GridCell

❌ ЕСЛИ GridCell НЕ ВНУТРИ Grid → GridManager их не найдет!

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #6: Консоль что говорит?
────────────────────────────────────────────────────────────────
❓ Есть ли сообщения в Console?

✅ ВКЛЮЧИТЬ DEBUG:
   1. Grid → Inspector → Grid Manager
   2. Show Debug: ✓ (включить)
   3. Play → кликайте
   4. Window → General → Console
   
📝 ЧТО ДОЛЖНО ПОЯВИТЬСЯ:
   "GridManager инициализирован. Найдено ячеек: X"
   При клике: "Размещен объект в ячейке..."

❌ ЕСЛИ ПИШЕТ:
   "GridManager не найден!" → см. Проверку #5
   "Префаб не назначен!" → см. Проверку #3
   "Ячейка уже занята!" → это нормально, клик на занятую
   
❌ ЕСЛИ НИЧЕГО НЕ ПИШЕТСЯ → клики не доходят (см. #1, #2, #4)

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА #7: Префаб имеет RectTransform?
────────────────────────────────────────────────────────────────
❓ У префаба который спавнится есть RectTransform?

✅ ПРОВЕРИТЬ:
   1. Project → Prefabs → ваш префаб
   2. Кликните на него
   3. Inspector → должен быть Rect Transform (НЕ обычный Transform!)

❌ ЕСЛИ обычный Transform → префаб не для UI!

🔧 КАК ИСПРАВИТЬ:
   1. Hierarchy → ПКМ → UI → Image
   2. Настройте его
   3. Перетащите в Prefabs
   4. Удалите из Hierarchy
