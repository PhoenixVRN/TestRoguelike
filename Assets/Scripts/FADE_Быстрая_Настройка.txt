1️⃣ СОЗДАТЬ ПАНЕЛЬ
───────────────────────────────────────────────────────────────

Canvas → ПКМ → UI → Image → "ScreenFadePanel"

Rect Transform:
  Anchors: Stretch-Stretch (покрывает весь экран)

Image:
  Color: Чёрный (0, 0, 0, 0) ← Alpha = 0!

Переместить В САМЫЙ ВЕРХ Canvas (последний в списке)

2️⃣ ДОБАВИТЬ КОМПОНЕНТ
───────────────────────────────────────────────────────────────

ScreenFadePanel → Add Component → Screen Fade

Settings:
  Fade In Duration: 0.5
  Fade Out Duration: 0.5
  Show Debug: ✓

3️⃣ ПОДКЛЮЧИТЬ
───────────────────────────────────────────────────────────────

BattleManager:
  Screen Fade: ScreenFadePanel ⭐

4️⃣ ТЕСТ
───────────────────────────────────────────────────────────────

Play → Победите врагов →
Экран темнеет 🌑 → Рестарт → Экран проясняется ☀️

✅ ГОТОВО!
