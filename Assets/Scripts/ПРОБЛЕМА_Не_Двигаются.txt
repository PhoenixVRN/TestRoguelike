    🛑 ПРОБЛЕМА: ПЕРСОНАЖИ НЕ ДВИГАЮТСЯ
═══════════════════════════════════════════════════════════════

Враги заспавнились, но стоят на месте? Давайте найдём причину!

═══════════════════════════════════════════════════════════════
🔍 ШАГ 1: ПРОВЕРЬТЕ CONSOLE
═══════════════════════════════════════════════════════════════

Посмотрите в Console после нажатия START:

✅ ХОРОШИЕ ЛОГИ (должно быть так):
┌─────────────────────────────────────────────────────────────┐
│ ⚔️ БОЙ НАЧАЛСЯ!                                            │
│ 📊 Всего героев на поле: X                                 │
│ ⚔️ Warrior вступил в бой!                                  │
│ ✅ Warrior готов к бою! [Team: 0] [HP: 100]               │
│ ⚔️ Enemy Warrior вступил в бой!                            │
│ ✅ Enemy Warrior готов к бою! [Team: 1] [HP: 80]          │
│ 📊 Команда 0 (игрок): 3 героев                            │
│ 📊 Команда 1 (враги): 3 героев    ← ДОЛЖНО БЫТЬ > 0!     │
│ ✅ Обе команды готовы! Бой начинается!                     │
└─────────────────────────────────────────────────────────────┘

❌ ПЛОХИЕ ЛОГИ (проблема):
┌─────────────────────────────────────────────────────────────┐
│ 📊 Команда 0 (игрок): 3 героев                            │
│ 📊 Команда 1 (враги): 0 героев    ← ПРОБЛЕМА!             │
│ ⚠️ Нет героев в команде 1 (враги)! Некого атаковать!     │
└─────────────────────────────────────────────────────────────┘

ИЛИ:
┌─────────────────────────────────────────────────────────────┐
│ ❌ У Enamy НЕТ CharacterConfig! Назначьте...              │
│ ❌ У Character НЕТ CharacterController! Добавьте...       │
└─────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════
🔍 ШАГ 2: ПРОВЕРЬТЕ TEAM У ВРАГОВ
═══════════════════════════════════════════════════════════════

ЭТО САМАЯ ЧАСТАЯ ПРИЧИНА!

1. Assets/Config/EnamyWarrior.asset
2. Откройте в Inspector
3. Проверьте поле Team:

   ❌ Team: 0  (НЕПРАВИЛЬНО! Будет союзником!)
   ✅ Team: 1  (ПРАВИЛЬНО! Враг!)

Если Team: 0 → измените на 1 и СОХРАНИТЕ!

═══════════════════════════════════════════════════════════════
🔍 ШАГ 3: ПРОВЕРЬТЕ ПРЕФАБ ВРАГА (Enamy)
═══════════════════════════════════════════════════════════════

1. Assets/Prefabs/Enamy.prefab
2. Откройте префаб (двойной клик)
3. Проверьте компоненты:

┌─────────────────────────────────────────────────────────────┐
│ Inspector                                                   │
│                                                             │
│ ✓ Rect Transform                                           │
│ ✓ Image                                                     │
│ ✓ Animator                                                  │
│ ✓ Character Animator (Script)                              │
│                                                             │
│ ✓ Character Controller (Script)  ← ДОЛЖЕН БЫТЬ!           │
│   ├─ Config: EnamyWarrior ✓      ← НАЗНАЧЕН!              │
│   ├─ Character Animator: ✓       ← НАЗНАЧЕН!              │
│   └─ Show Debug: ✓               ← ВКЛЮЧИТЬ!              │
└─────────────────────────────────────────────────────────────┘

Если Character Controller НЕТ:
  → Add Component → Character Controller

Если Config пустой (None):
  → Перетащите EnamyWarrior.asset в поле Config

Если Character Animator пустой:
  → Перетащите компонент Character Animator с этого же объекта

СОХРАНИТЕ префаб! (Apply All)

═══════════════════════════════════════════════════════════════
🔍 ШАГ 4: ПРОВЕРЬТЕ ПРЕФАБ ГЕРОЯ (Character)
═══════════════════════════════════════════════════════════════

То же самое для героя:

1. Assets/Prefabs/Character.prefab
2. Character Controller:
   ├─ Config: Warrior ✓       ← Team: 0
   ├─ Character Animator: ✓
   └─ Show Debug: ✓

═══════════════════════════════════════════════════════════════
🔍 ШАГ 5: ТЕСТ С ОТЛАДКОЙ
═══════════════════════════════════════════════════════════════

Включите Show Debug везде:

1. GameManager → Show Debug: ✓
2. WaveSpawner → Show Debug: ✓
3. Character (префаб) → CharacterController → Show Debug: ✓
4. Enamy (префаб) → CharacterController → Show Debug: ✓

Запустите снова и смотрите Console!

Если персонажи инициализируются правильно, увидите:
┌─────────────────────────────────────────────────────────────┐
│ ✅ Warrior инициализирован. HP: 100                        │
│ ⚔️ Warrior вступил в бой!                                  │
│ 🎯 Warrior нашёл цель: Enemy Warrior                       │
│ ⚔️ Warrior атаковал Enemy Warrior на 10 урона!            │
└─────────────────────────────────────────────────────────────┘

Если НЕ видите таких логов → персонажи не инициализируются!

═══════════════════════════════════════════════════════════════
❌ ЧАСТЫЕ ОШИБКИ И РЕШЕНИЯ
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА 1: "Команда 1 (враги): 0 героев"
ПРИЧИНА: Враги заспавнились, но у них Team: 0 (как у героев)
РЕШЕНИЕ: EnamyWarrior.asset → Team: 1

ПРОБЛЕМА 2: "У Enamy НЕТ CharacterConfig"
ПРИЧИНА: Префаб врага не настроен
РЕШЕНИЕ: Enamy.prefab → CharacterController → Config: EnamyWarrior

ПРОБЛЕМА 3: Персонажи стоят, ничего не делают
ПРИЧИНА: CharacterController.StartBattle() не вызван
РЕШЕНИЕ: Проверьте GameManager вызывает StartBattle()

ПРОБЛЕМА 4: NullReferenceException
ПРИЧИНА: Character Animator не назначен
РЕШЕНИЕ: CharacterController → Character Animator: назначить

ПРОБЛЕМА 5: "Warrior нашёл цель: Unknown"
ПРИЧИНА: У цели нет конфига
РЕШЕНИЕ: Убедитесь что у ВСЕХ персонажей есть Config

═══════════════════════════════════════════════════════════════
🧪 БЫСТРЫЙ ТЕСТ
═══════════════════════════════════════════════════════════════

1. Запустите игру (Play)
2. Поставьте ОДНОГО героя слева
3. Нажмите START
4. Смотрите Console:

ДОЛЖНО БЫТЬ:
  ✅ Warrior инициализирован
  ⚔️ Warrior вступил в бой
  ✅ Enemy Warrior инициализирован
  ⚔️ Enemy Warrior вступил в бой
  📊 Команда 0: 1 героев
  📊 Команда 1: X героев  ← НЕ НОЛЬ!
  🎯 Warrior нашёл цель: Enemy Warrior
  (через секунду)
  ⚔️ Warrior атаковал Enemy Warrior

Если видите ЭТО → система работает!
Если НЕТ → смотрите что именно отсутствует в логах

═══════════════════════════════════════════════════════════════
📋 ЧЕКЛИСТ "ВСЁ ЛИ НАСТРОЕНО?"
═══════════════════════════════════════════════════════════════

□ EnamyWarrior.asset → Team: 1 (НЕ 0!)
□ Warrior.asset → Team: 0
□ Enamy.prefab → CharacterController есть
□ Enamy.prefab → Config: EnamyWarrior назначен
□ Enamy.prefab → Character Animator назначен
□ Character.prefab → CharacterController есть
□ Character.prefab → Config: Warrior назначен
□ Character.prefab → Character Animator назначен
□ Show Debug включён везде
□ После START в Console есть "⚔️ БОЙ НАЧАЛСЯ!"
□ В Console есть "Команда 1 (враги): X героев" где X > 0

Если ВСЁ ✓ но не работает → скопируйте логи из Console!
