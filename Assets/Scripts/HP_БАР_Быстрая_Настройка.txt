📝 ШАГ 1: СОЗДАТЬ UI СТРУКТУРУ
───────────────────────────────────────────────────────────────

Откройте Assets/Prefabs/Character.prefab

В Hierarchy создайте:

Character
└── HealthBarCanvas (UI → Canvas)
    └── HealthBar (UI → Image)
        ├── Background (UI → Image, чёрный)
        └── Fill (UI → Image, зелёный, Type: Filled)

═══════════════════════════════════════════════════════════════
📝 ШАГ 2: НАСТРОИТЬ HealthBarCanvas
───────────────────────────────────────────────────────────────

HealthBarCanvas → Inspector:
  Canvas → Render Mode: World Space
  Rect Transform:
    Pos Y: 60
    Width: 80
    Height: 10
    Scale: (0.1, 0.1, 1)

═══════════════════════════════════════════════════════════════
📝 ШАГ 3: НАСТРОИТЬ Fill
───────────────────────────────────────────────────────────────

Fill → Inspector:
  Image Type: Filled
  Fill Method: Horizontal
  Fill Origin: Left
  Color: Зелёный

═══════════════════════════════════════════════════════════════
📝 ШАГ 4: ДОБАВИТЬ КОМПОНЕНТ
───────────────────────────────────────────────────────────────

HealthBar → Add Component → Health Bar

Health Bar:
  Fill Image: Fill (перетащить)
  Background Image: Background
  Smooth Transition: ✓

═══════════════════════════════════════════════════════════════
📝 ШАГ 5: ПОДКЛЮЧИТЬ К ПЕРСОНАЖУ
───────────────────────────────────────────────────────────────

Character → CharacterController:
  Health Bar: HealthBar (перетащить)

СОХРАНИТЬ ПРЕФАБ! (Apply All)

═══════════════════════════════════════════════════════════════
📝 ШАГ 6: ПОВТОРИТЬ ДЛЯ ВРАГА
───────────────────────────────────────────────────────────────

Enamy.prefab → скопировать HealthBarCanvas из Character
Назначить в CharacterController

═══════════════════════════════════════════════════════════════
✅ ГОТОВО!
═══════════════════════════════════════════════════════════════

Запустите игру → HP бары появятся автоматически! ❤️
