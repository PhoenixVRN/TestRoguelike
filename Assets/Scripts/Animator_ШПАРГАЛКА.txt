═══════════════════════════════════════════════════════════════
    ⚡ ANIMATOR - БЫСТРАЯ НАСТРОЙКА (BOOL)
═══════════════════════════════════════════════════════════════

Character.controller → Откройте

═══════════════════════════════════════════════════════════════
1️⃣ PARAMETERS
═══════════════════════════════════════════════════════════════

Создайте 3 Bool:
  + IsMoving (Bool)
  + IsAttacking (Bool)
  + IsDead (Bool)

Удалите все старые Trigger!

═══════════════════════════════════════════════════════════════
2️⃣ STATES
═══════════════════════════════════════════════════════════════

Idle (default) → Motion: IdleCharacter
Move → Motion: MoveCharacter
Attack → Motion: AttackCharacter (Loop Time: ✓)
Death → Motion: DeathCharacter (Loop Time: ☐)

═══════════════════════════════════════════════════════════════
3️⃣ TRANSITIONS (переходы)
═══════════════════════════════════════════════════════════════

Idle → Move:
  Condition: IsMoving = true
  Has Exit Time: ☐

Move → Idle:
  Condition: IsMoving = false
  Has Exit Time: ☐

Idle → Attack:
  Condition: IsAttacking = true
  Has Exit Time: ☐

Move → Attack:
  Condition: IsAttacking = true
  Has Exit Time: ☐

Attack → Idle:
  Condition: IsAttacking = false
  Has Exit Time: ☐

Idle → Death:
  Condition: IsDead = true
  Has Exit Time: ☐

Move → Death:
  Condition: IsDead = true
  Has Exit Time: ☐

Attack → Death:
  Condition: IsDead = true
  Has Exit Time: ☐

═══════════════════════════════════════════════════════════════
4️⃣ ВАЖНО
═══════════════════════════════════════════════════════════════

✓ AttackCharacter.anim → Loop Time: ✓ (зациклена)
☐ DeathCharacter.anim → Loop Time: ☐ (НЕ зациклена)

Ctrl+S (сохранить)

═══════════════════════════════════════════════════════════════
✅ ГОТОВО!
═══════════════════════════════════════════════════════════════

Play → Атаки зациклены! Смерть останавливает всё! ⚔️💀
